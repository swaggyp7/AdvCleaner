## Features

- 屏蔽百度搜索内容，侧边栏，底部广告。
- 屏蔽百度知道内容，侧边栏，底部广告。
- 屏蔽百度贴吧侧边，贴文广告。
- 屏蔽百度新闻评论区，净化眼睛。
- 屏蔽CSDN侧边广告，免登录复制内容。
- 屏蔽知乎首页，侧边广告。
- 屏蔽知乎盐选会员广告。
- 屏蔽掘金插件推广，侧边广告。
- 屏蔽简书第三方广告。
